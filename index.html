What is STVR Doorman?

STVR Doorman is a free and open-source platform for municipalities, counties, HOAs, and community groups to monitor, map, and manage Short-Term Vacation Rentals (STVRs). It brings together:

  • Interactive maps of licensed STVR properties
  • Links to Airbnb, VRBO, and other platform listings
  • Public complaint submission and visibility via Google Forms linked to automated update scripts
  • Customizable levels of transparency for public reporting and data sharing
  • Occupancy limit calculations and over-limit detection
  • Municipal boundary and parcel-based approximate address mapping

The platform features customizable levels of transparency so communities can tailor reporting to local policies. Complaints can be configured to be entirely public by default, moderated before publishing, made public only after official verification, or kept private for internal code enforcement use only.

The platform uses open tools like MapLibre GL, GeoJSON, Shapefiles, and Google Forms. Hosting options are available entirely for free on GitHub Pages and GitHub repositories. With the right data files—municipal boundaries, parcel data, and a list of licensed STVRs—any jurisdiction can deploy a fully functional STVR map and complaint portal at zero cost.

Open data & script automation example: Municipalities can use open GIS portals such as https://data-sagis.opendata.arcgis.com/ to obtain boundary and parcel data. Public complaints submitted through Google Forms run a lightweight backend script that automatically processes entries so the complaints map and data layers update seamlessly according to your chosen transparency settings.

STVR Doorman is designed so that any municipality can revert to the free tier at any time. Paid tiers add setup, customization, hosting, automation, and enforcement tools—but the core platform always remains free and open.

Community support is available via a public GitHub repository for bug reports and feature requests. No direct support is included in the free tier, but the code and documentation are open to everyone.

Tier Details

Free Tier – Open Source, Zero Cost

Best for: Cities with technical staff or GIS capability.

Includes:

  • Full STVR mapping using MapLibre GL
  • Display of licensed properties and platform links (Airbnb, VRBO, etc.)
  • Public-facing portal with a Google Form for complaints tied to an automated update script
  • Configurable transparency settings (public by default, moderated, or verified-only)
  • Free hosting options available directly on GitHub
  • Unlimited properties and submissions
  • Community support via GitHub

You provide your own data files (municipal boundaries, parcel data, and license list). With the correct shape/GeoJSON files and STVR locations, the entire map can be populated and hosted publicly at zero cost.

Tier 1 – Assisted Setup (Self-Hosted)

Best for: Municipalities that want the free platform but need help setting it up.

Includes everything in Free, plus:

  • One-time setup of the free platform
  • Integration of your license list, boundaries, and parcel data
  • Integration of your Google Form and script pipeline with tailored transparency rules
  • Unlimited properties

You host and maintain the platform yourself (with free options on GitHub). Additional work is available at a billable rate of $200/hr.

Tier 2 – Custom Complaint System (Self-Hosted)

Best for: Cities wanting a custom complaint workflow.

Includes everything in Tier 1, plus:

  • Fully custom complaint form (not limited to Google Forms)
  • Complaint visualization directly on the map via automated data scripts
  • Flexible transparency controls (moderation queues, public-by-default, or verification gates)
  • Integration into your existing web infrastructure

You still host and maintain the platform yourself, with optional paid support at $200/hr.

Tier 3 – Managed Hosting (Up to 5,000 Properties)

Best for: Cities wanting a turnkey hosted solution.

Includes everything in Tier 2, plus:

  • Fully hosted platform managed by STVR Doorman
  • Twice-yearly updates
  • Up to 20 hours of maintenance and repairs per year
  • Priority support
  • Up to 5,000 properties

If geographic files are incomplete or malformed, additional fees may apply for data cleanup.

Tier 4 – Real-Time Notifications (Up to 10,000 Properties)

Best for: Cities needing enforcement-grade tools.

Includes everything in Tier 3, plus:

  • Real-time push notifications to hosts
  • Real-time alerts to enforcement officials (e.g., 911 dispatch, code enforcement)
  • No geographic files required—only a list of licenses
  • Up to 10,000 properties

You can revert to the free tier at any time and continue using a Google Form-based complaint portal.

Tier 5 – Ultra Premium (Up to 15,000 Properties)

Best for: Cities needing full STVR discovery and population.

Includes everything in Tier 4, plus:

  • No initial license list required
  • Manual identification and seeding of up to 15,000 STVR properties
  • Annual re-check for new properties
  • Up to 40 hours of custom requests and configuration per year
  • All features from all previous tiers

This tier is a complete STVR compliance solution requiring minimal municipal effort.

Community Contributions & Visitor Submissions

Regardless of tier, visitors can always add properties by clicking on the map and submitting links to listings on platforms like Airbnb, VRBO, and others. This helps keep the map current and allows communities to participate directly in monitoring short-term rentals.

The project will maintain a public GitHub repository where users can:

  • Report bugs
  • Request features
  • Contribute improvements
  • Share configuration examples
